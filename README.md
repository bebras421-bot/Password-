# Password Manager Console Application

**Author:** Попова Мария

## Description
Console password manager with JSON storage, password generation, and MVC architecture.

## Features
- Add, view, and delete password records
- Generate random passwords with custom settings (length 8-64, digits/letters/symbols)
- Data persistence using JSON
- Input validation
- MVC pattern implementation
- Full OOP principles (encapsulation, inheritance, polymorphism)

## Requirements
- Python 3.6+

git clone <your-repo-url>
